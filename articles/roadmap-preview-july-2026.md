The Fleet roadmap is set for summer 2026. Watch the video above for a walkthrough, or continue reading for the highlights.

In the next 3 months, Fleet will ship...

- 🩹 Patch policies: Set a deadline, prompt end users before it hits, install when the app is closed, and patch during maintenance windows ([#39176](https://github.com/fleetdm/fleet/issues/39176), [#39178](https://github.com/fleetdm/fleet/issues/39178), [#39962](https://github.com/fleetdm/fleet/issues/39962), [#48174](https://github.com/fleetdm/fleet/issues/48174))
- 🍏 Enroll and manage tvOS ([#38791](https://github.com/fleetdm/fleet/issues/38791))
- 📱 iOS/iPadOS: Auto-install apps after enrollment ([#38789](https://github.com/fleetdm/fleet/issues/38789))
- 🪟 Windows: Create, force standard, and rotate local admin accounts ([#43488](https://github.com/fleetdm/fleet/issues/43488), [#43489](https://github.com/fleetdm/fleet/issues/43489), [#43490](https://github.com/fleetdm/fleet/issues/43490))
- 🔄 Staged rollouts for Fleet-maintained apps ([#48090](https://github.com/fleetdm/fleet/issues/48090))
- 📋 Configuration profiles: Upload ADMX (Windows), DDM declarations and assets, and JSON exports from Intune — and edit them in place ([#48103](https://github.com/fleetdm/fleet/issues/48103), [#48222](https://github.com/fleetdm/fleet/issues/48222), [#48198](https://github.com/fleetdm/fleet/issues/48198), [#48046](https://github.com/fleetdm/fleet/issues/48046))
- 🔐 View certificates on host details for macOS, Windows, and Android ([#39346](https://github.com/fleetdm/fleet/issues/39346), [#31294](https://github.com/fleetdm/fleet/issues/31294), [#43315](https://github.com/fleetdm/fleet/issues/43315))
- 📜 CIS Benchmarks: Update macOS 15 and 14, update Windows 10 to v4.0.0 ([#45644](https://github.com/fleetdm/fleet/issues/45644), [#35118](https://github.com/fleetdm/fleet/issues/35118))
- 🛒 App Store (VPP): Uninstall apps and revoke licenses ([#25497](https://github.com/fleetdm/fleet/issues/25497), [#25912](https://github.com/fleetdm/fleet/issues/25912))
- ⏰ OS updates: Trigger from critical CVEs, set a date/time deadline, and update within a minor version ([#45605](https://github.com/fleetdm/fleet/issues/45605), [#47188](https://github.com/fleetdm/fleet/issues/47188), [#45511](https://github.com/fleetdm/fleet/issues/45511))
- 🧩 Custom host vitals: Pull from your IdP across all platforms and use built-in vitals to define labels ([#42922](https://github.com/fleetdm/fleet/issues/42922), [#47956](https://github.com/fleetdm/fleet/issues/47956))
- 🔍 Filter vulnerable software by CVSS severity ([#47326](https://github.com/fleetdm/fleet/issues/47326))
- 🔔 Webhooks for host activities ([#40493](https://github.com/fleetdm/fleet/issues/40493))
- 📢 Policy automations: Send a Slack message or re-send a configuration profile ([#44436](https://github.com/fleetdm/fleet/issues/44436), [#40637](https://github.com/fleetdm/fleet/issues/40637))
- 🤖 Android: Run custom MDM commands and view them in the CLI, API, and host details ([#23232](https://github.com/fleetdm/fleet/issues/23232), [#33158](https://github.com/fleetdm/fleet/issues/33158))
- 📦 Software inventory: Go binaries, VS Code extensions, Linux apps, and Adobe plugins ([#44775](https://github.com/fleetdm/fleet/issues/44775), [#47790](https://github.com/fleetdm/fleet/issues/47790), [#47789](https://github.com/fleetdm/fleet/issues/47789), [#45414](https://github.com/fleetdm/fleet/issues/45414))
- ✨ Fleet's MCP server: Remove the experimental tag ([#44448](https://github.com/fleetdm/fleet/issues/44448))

Big opportunities that Fleet is building towards in the near future (next 180 days):

- 🔖 New CIS and custom policy types ([#48036](https://github.com/fleetdm/fleet/issues/48036), [#48037](https://github.com/fleetdm/fleet/issues/48037))
- 💻 Deploy Microsoft Store apps to Windows ([#43493](https://github.com/fleetdm/fleet/issues/43493))
- 🤖 Android: Auto-install apps after enrollment ([#40689](https://github.com/fleetdm/fleet/issues/40689))
- 🏷️ Software: Custom targets with "Include any" and "Exclude any" ([#33443](https://github.com/fleetdm/fleet/issues/33443))
- ⚡ Automatically install software on the right architecture for each host (ARM vs. x64) ([#47840](https://github.com/fleetdm/fleet/issues/47840))

Any feedback or questions? Contributions welcome! You can find us [where we hang out](https://fleetdm.com/support).
