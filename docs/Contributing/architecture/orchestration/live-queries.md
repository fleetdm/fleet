# Live reports

Live reports in Fleet allow users to execute ad-hoc reports against devices in real-time, providing immediate visibility into device status, configuration, and security posture. This document provides insights into the design decisions, system components, and interactions specific to the Live Reports functionality.

See our [live reports guide](https://fleetdm.com/guides/get-current-telemetry-from-your-devices-with-live-queries) to learn more about the architecture, implementation, and use of live reports.
