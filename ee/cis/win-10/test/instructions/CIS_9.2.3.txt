Expected scenario
==================
1) Open "Edit Group Policy" tool and set the following UI path to 'Allow (default)':
    'Computer Configuration\Policies\Windows Settings\Security Settings\Windows Defender Firewall with Advanced Security\Windows Defender Firewall with Advanced Security - Local Group Policy Object\Windows Defender Firewall Properties\Private Profile\Outbound connections'
2) After running the policy check, it should return 1 indicating that setting was properly set



Failure scenario
==================
1) Open "Edit Group Policy" tool and set the following UI path to values different than 'Allow (default)':
    'Computer Configuration\Policies\Windows Settings\Security Settings\Windows Defender Firewall with Advanced Security\Windows Defender Firewall with Advanced Security - Local Group Policy Object\Windows Defender Firewall Properties\Private Profile\Outbound connections'

2) After running the policy check, it should return nothing, indicating that setting was set to a non-compliant value
