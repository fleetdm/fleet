# Slack

## What is Slack?

Slack is Fleet's primary internal communication platform used for team collaboration, project discussions, and real-time messaging. It serves as the central hub for team communication, integrates with many of Fleet's tools, and supports both synchronous and asynchronous work.

## Systems it integrates with

Slack integrates with numerous systems and tools at Fleet:

- **GitHub** - Repository updates, pull requests, and issue notifications
- **Salesforce** - Opportunity and account updates
- **Gong.io** - Call recordings and insights
- **Zapier** - Various automation workflows
- **Google Calendar** - Meeting reminders and calendar integration
- **1Password** - Secure credential sharing
- **Various bots and tools** - Custom integrations for notifications and workflows
